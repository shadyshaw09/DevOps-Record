Day 12 - Docker Installation and Setup
Date: 21-02-2026

Objective: To install Docker Desktop and verify its setup.

Introduction: Docker Desktop is an application that allows users to run Docker on Windows or macOS systems. It provides a graphical interface and includes Docker Engine, Docker CLI, and Docker Compose.

Installation Steps Performed:
1. Downloaded Docker Desktop from the official website.
2. Installed Docker Desktop on Windows.
3. Enabled WSL 2 integration.
4. Restarted the system after installation.
5. Launched Docker Desktop successfully.

Verification Commands Practiced:
docker --version  
docker info  
docker help  

These commands confirmed that Docker was installed successfully.

Docker Desktop Interface:

Docker Desktop provides:
- Containers tab
- Images tab
- Volumes
- Networks

We explored the dashboard and understood how to manage containers.

Importance in DevOps:
- Required for container-based development
- Essential for building and running images
- Used for microservices deployment

Conclusion:
Today I installed Docker Desktop and verified the installation using Docker CLI commands.
